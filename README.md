# AI Feeds

Personal AI industry intelligence system. Aggregates signals from papers, trending repos, news, and community discussions into actionable briefings.

## What It Does

Pulls from multiple AI information sources, filters for relevance to your learning plan and work context, and generates Obsidian-compatible signal snapshots.

### Feed Sources (Planned)
- **Papers**: arxiv, Semantic Scholar, HuggingFace Daily Papers
- **Code**: GitHub trending, awesome lists, OSS Insight
- **News**: AI newsletters (The Batch, Import AI, AI Weekly), HN, Reddit
- **Industry**: Product Hunt AI, funding rounds, job market signals
- **Social**: Twitter/X AI lists, Discord communities

### Output
- Daily/weekly signal snapshots in Obsidian markdown
- Relevance-scored against your learning plan
- Knowledge gap detection and learning priority recommendations
